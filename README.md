# 🏧 ATM Mini Project

![ATM Machine](https://images.unsplash.com/photo-1601597111158-2fceff292cdc?auto=format&fit=crop&q=80)

## 📌 Overview

This ATM simulation project allows users to perform essential banking operations with a secure and user-friendly interface.

## 🎯 Key Features


### 🔐 1. User Authentication
- Secure login with **Account Number** and **PIN**
- Multi-layer security verification

### 💰 2. Account Operations
- **Check Balance** - View your current balance instantly
- **Cash Withdrawal** - Quick and secure cash withdrawals
- **Show User Details** - Access your account information
- **Update Mobile Number** - Easy profile management

- Implements secure class-based architecture
- Features private member variables for data protection
- Includes robust error handling and input validation

## 🔄 How It Works

1. **Login** → Enter Account Number and PIN
2. **Access** → View main menu with multiple options
3. **Operate** → Perform desired banking operations
4. **Secure** → Auto-logout for security

## 🚀 Getting Started

Simply compile and run the program to start using the ATM simulation system. Follow the on-screen prompts to navigate through different features.
